## 开启Telnet登录

默认是关闭了Telnet功能的，连接上光猫后访问以此地址来开启Telnet功能：

http://192.168.1.1/usr=CMCCAdmin&psw=aDm8H%25MdA&cmd=1&telnet.gch

看到以下信息后说明开启成功。

![image-20250909114150406](/Users/asins/Library/Application Support/typora-user-images/image-20250909114150406.png)

使用 telnet 登录光猫，**注意密码不显示，不要重复输入**

> telnet 192.168.1.1

​    账号：CMCCAdmin
​    密码：aDm8H%MdA

然后就可以查看web登录账号密码，可以看到密码为加密显示

> sidbg 1 DB p DevAuthInfo

执行以下命令修改CMCCAdmin的登陆密码为：aDm8H%MdA

> sidbg 1 DB set DevAuthInfo 0 Pass aDm8H%MdA

执行以下命令保存密码的修改。

> sidbg 1 DB save 

然后就可以在`http://192.168.1.1/`网页中使用用户名为 `CMCCAdmin` 密码为`aDm8H%MdA` 来登录路由器。



```bash
# 先下载，再编辑模式查看密码
sidbg 1 DB decry /userconfig/cfg/db_user_cfg.xml
cd /tmp

# 再搜索/DevAuthInfo 就能看到密码了
```
